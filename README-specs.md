# README Specs

Этот каталог описывает правила разработки проекта **OKRs** в формате spec-based development.

Цель этих документов — сократить число итераций при работе с AI и сделать изменения в проекте более предсказуемыми:

- каждая существенная фича сначала описывается в спецификации;
- AI реализует изменения только в рамках явного scope;
- архитектурные, доменные и продуктовые ограничения фиксируются отдельно от кода;
- любая новая функциональность опирается на единый шаблон spec-документа.
