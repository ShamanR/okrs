# Product Overview

## Назначение

Система предназначена для ведения OKR нескольких команд внутри общей оргструктуры.
Базовая сущность просмотра — команда в рамках периода.

Система должна позволять:

- видеть иерархию команд;
- просматривать OKR команды за выбранный период;
- создавать цели и KR;
- обновлять прогресс KR;
- комментировать цели и KR;
- расшаривать одну цель на несколько команд;
- управлять статусом периода для команды.

## Что уже считается частью продукта

В текущем проекте уже существуют:

- team hierarchy;
- periods;
- goals;
- key results типов PROJECT, PERCENT, LINEAR, BOOLEAN;
- comments;
- team period statuses no_goals, forming, in_progress, validated, closed;
- shared goals через goal_shares.

## Ценность продукта

Продукт решает три задачи:

1. Делает OKR прозрачными по оргструктуре.
2. Позволяет фиксировать прогресс в едином формате.
3. Даёт единый контур для owner team и shared team при работе с общей целью.
